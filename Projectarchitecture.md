# Folder Structure:

Use self contained folders based on routes, common ui components and separated custom components

## Example:
```
/app
  /containers (separated pages or routes)
    home.js
    transaction.js
  /ui-common
    button.js
    input.js
    form.js
    table.js
  /components
    /component
      index.js (HOC)
      component1.js
      component2.js
      component.css
      component.tes.js
  /actions
  /reducers
```
